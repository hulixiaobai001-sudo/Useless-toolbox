## 🇨🇳 简体中文

> 一个集成了 7 个看起来很专业、用起来很快乐、但实际上完全没用的工具的纯前端工具箱。双击 HTML 即可运行，无需安装任何东西。

### ✨ 已收录的无用工具

| # | 工具 | 描述 |
|---|------|------|
| 1 | 🦊 狐狸语翻译器 | 把任何话翻译成狐狸能听懂的语言 |
| 2 | ⏳ 假装在加载 | 一个永远到不了 100% 的进度条 |
| 3 | 📅 今日不宜 | 随机告诉你今天不该做什么（别当真） |
| 4 | ⏰ 焦虑倒计时 | 精确到秒地告诉你明天还有多远 |
| 5 | 💡 人生建议生成器 | 给你一条完全没用的人生建议 |
| 6 | 🔮 灵魂鉴定器 | 生成一份看起来很专业的灵魂分析报告 |
| 7 | 💻 假装很忙模拟器 | 让终端疯狂滚动，看起来在编译重要项目 |

### 🚀 快速开始

1. 下载 `index.html`
2. 双击用浏览器打开
3. 开始浪费时间

就这么多。不需要 Python，不需要 Node.js，不需要 `npm install`。它就是一个 HTML 文件。

### 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript（零依赖，零框架）
- 终端风格 UI + CRT 扫描线效果
- 单文件，离线可用

### 📁 项目结构

```
Useless-toolbox/
├── index.html          # 整个项目就这一个文件
├── README.md           # 你正在看的这个
├── LICENSE             # MIT 许可证
└── ADDITIONAL_LICENSE.md  # 附加协议（含狐狸条款）
```

### 🧩 添加你自己的无用工具

打开 `index.html`，找到 JavaScript 部分，用 `register()` 注册新工具：

```javascript
register('我的无用工具', '一句话描述它有多没用', '🎯', async () => {
  // 在这里写你的无用逻辑
});
```

完成。你的工具会自动出现在主菜单中。

## 🇺🇸 English

> A pure frontend toolbox integrating 7 tools that look professional, feel fun to use, but are completely useless. Just open the HTML file in a browser — no installation required.

### ✨ Included Useless Tools

| # | Tool | Description |
|---|------|-------------|
| 1 | 🦊 Fox Translator | Translate anything into fox language |
| 2 | ⏳ Fake Loading | A progress bar that never reaches 100% |
| 3 | 📅 Don't Do Today | Randomly tells you what not to do today |
| 4 | ⏰ Anxiety Timer | Counts down to tomorrow, to the centisecond |
| 5 | 💡 Life Advice Generator | Gives you a completely useless piece of life advice |
| 6 | 🔮 Soul Analyzer | Generates a professional-looking soul analysis report |
| 7 | 💻 Busy Simulator | Makes your terminal scroll wildly, looking like you're compiling something important |

### 🚀 Quick Start

1. Download `index.html`
2. Open it in a browser
3. Start wasting time

That's it. No Python, no Node.js, no `npm install`. It's just one HTML file.

### 🛠️ Tech Stack

- Pure HTML + CSS + JavaScript (zero dependencies, zero frameworks)
- Terminal-style UI + CRT scanline effect
- Single file, works offline

### 📁 Project Structure

```
Useless-toolbox/
├── index.html             # The entire project in one file
├── README.md              # What you're reading now
├── LICENSE                # MIT License
└── ADDITIONAL_LICENSE.md  # Additional License (includes Fox Clause)
```

### 🧩 Add Your Own Useless Tool

Open `index.html`, find the JavaScript section, and register a new tool with `register()`:

```javascript
register('My Useless Tool', 'One sentence on how useless it is', '🎯', async () => {
  // Write your useless logic here
});
```

Done. Your tool will automatically appear in the main menu.
